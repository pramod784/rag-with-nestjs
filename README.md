
# NestJS Multi-Tenant RAG Enterprise

## Features
- Azure AI Search Hybrid (Vector + Semantic)
- Multi-tenant filtering
- Intent detection
- Rule engine (latest logic)
- LLM reranking
- Redis caching
- Azure index schema included
